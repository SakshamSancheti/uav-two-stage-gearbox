# Two-Stage Reduction Gearbox for UAV Thrust Maximization

This project involves the complete design and analysis of a lightweight two-stage reduction gearbox for UAV propulsion. The design focuses on maximizing thrust efficiency while keeping the gearbox compact, lightweight, and manufacturable.
